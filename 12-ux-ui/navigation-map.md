# Mapa de navegación

> Estado: 🟡 En progreso | Última actualización: 2026-06-24
> Autor: Equipo de UX/UI | Equipo: Diseño

## Navegación principal

- **Dashboard**
- **Fichas**
- **Ambientes**
- **Instructores**
- **Horarios**
- **Documentos**
- **Alertas y notificaciones**
- **Configuración / Administración**

## Flujo de usuarios

- Coordinador: desde Dashboard → Fichas → Crear/Editar ficha → Asignar horario.
- Instructor: Dashboard → Mis horarios → Detalle de sesión.
- Aprendiz: Dashboard → Mi ficha → Descargar documento.
- Administrador: Configuración → Usuarios → Roles.
